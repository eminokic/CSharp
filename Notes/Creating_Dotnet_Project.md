# Step 1  Creating Foundation

$ cd Workspace <br/>
$ mkdir project <br/>
$ cd project <br/>
$ mkdir src <br/>
$ mkdir test <br/>

## Step 2 Creating New Dotnet Console

$ cd src <br/>
$ mkdir project (but with slightly different name) <br/>
$ cd /src/project <br/>
$ dotnet new console <br/>


## Step 3 Run The .Net Project

$ dotnet run <br/>
Should originally output: <br/>
```bash
"Hello World!"
```
